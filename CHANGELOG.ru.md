# История изменений

## [1.0.0] — 2026-04-06

Первый стабильный релиз.

### Новые возможности
- **Хранилище паролей** — логины, карты, защищённые заметки, SSH-ключи
- **Zero-knowledge шифрование** — сервер никогда не видит ваши пароли; все данные шифруются на стороне клиента (AES-256-GCM)
- **Двухфакторная аутентификация** — TOTP (Google Authenticator, Authy, 2FAS)
- **Passkeys** — вход через Face ID, Touch ID, YubiKey или Windows Hello
- **Ключ восстановления** — резервный доступ к хранилищу при утере мастер-пароля
- **Генератор паролей** — пароли, парольные фразы (EFF-словарь), PIN-коды с показателем энтропии
- **Импорт** — Keylone JSON, Bitwarden JSON, Chrome/Edge, Firefox, Safari, 1Password, LastPass, KeePass XML, Dashlane, NordPass
- **Экспорт** — Keylone JSON с опциональной защитой паролем
- **Проверка утечек** — анонимная проверка паролей по базе HaveIBeenPwned
- **Папки** — неограниченная вложенность, перетаскивание
- **Вложения** — прикрепление файлов к записям (до 10 МБ, шифруются)
- **Telegram-уведомления** — оповещения о входах и ссылки сброса пароля через Telegram-бот
- **Панель администратора** — управление пользователями, контроль регистрации, политика истории паролей, настройка SMTP/Telegram
- **Браузерные расширения** — Chrome, Firefox, Safari с автозаполнением
- **Десктопное приложение** — macOS и Windows (Tauri)
- **Тёмная / светлая тема**
- **Русский и английский интерфейс** — язык определяется автоматически по настройкам браузера

### Инфраструктура
- Готовый Docker-образ `ghcr.io/yukh975/keylone:latest`
- Встроенный PostgreSQL (быстрый старт) и поддержка внешнего PostgreSQL
- Автоматическая сборка образа при каждом релизе через GitHub Actions
- `setup.sh` / `setup.external.sh` — развёртывание одной командой с автогенерацией секретов
- `update.sh` — обновление образа без затрагивания базы данных
- `backup.sh` — резервная копия базы данных с инструкцией по восстановлению
